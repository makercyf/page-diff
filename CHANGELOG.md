# Change Log

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/)
and this project adheres to [Semantic Versioning](http://semver.org/).

## [1.0.0] - 2026-07-21

### Added

- Content-based image comparison across two to four folders.
- PHash, DHash, WHash, and AHash comparison methods.
- Detection of missing, extra, duplicate, and blank pages.
- Live comparison progress using Server-Sent Events.
- Adjustable similarity threshold and blank-page sensitivity.
- Synchronized and independent scrolling modes.
- Windows startup script that launches the server and opens the app.
- Installation support through `uv` and `pip`.
